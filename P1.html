Translation / Rotation


PROPERTY DESCRIPTION IMAGE
Translation The object begins to translate at 45 degrees once the key 'k' is pressed. The corners of the object are calculated and it follows elastic collision with the boundaries of the wall
Methods Involved: translateVelocity(),
translate(), rebound()
Rotation The object rotates about its Centre Of Mass when the ket 'r' is pressed.
Methods Involved: rotateVelocity(), rotate()
Particle is a class which contains the above mentioned methods. The object of the class is created and the methods are called for the same.
